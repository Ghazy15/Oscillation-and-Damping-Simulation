# Oscilation-and-Damper-Simulation
A web-based simulation project application to simulate a real oscilation and damper situation, The App were made with Python 3.12.10 and JavaScript

To run the App from a fresh Python evironment, do:
1. Create a Virtual Environment (if not done yet), with 'python -m venv .venv'
2. Active the Venv, in Windows '.venv\Scripts\activate'
3. Install all the dependencies, with 'pip install -r requirements.txt'
4. Run the server, with 'uvicorn main:app --reload'
5. Open the Web-apps, through 'http://localhost:8000/static/index.html'
