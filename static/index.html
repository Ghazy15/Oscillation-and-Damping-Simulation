Dasar Teori

Hukum Newton II untuk sistem massa-pegas dengan redaman menghasilkan persamaan diferensial orde dua:

$$m\ddot{x} + c\dot{x} + kx = 0$$

Solusi umum bergantung pada nilai rasio redaman:

$$\zeta = \frac{c}{c_c} = \frac{c}{2\sqrt{mk}}$$

Tanpa Redaman ζ = 0

$$x(t) = x_0\cos(\omega_n t) + \frac{v_0}{\omega_n}\sin(\omega_n t)$$

Sistem berosilasi terus-menerus tanpa kehilangan energi.

Underdamped ζ < 1

$$x(t) = e^{-\zeta\omega_n t}\left[A\cos(\omega_d t) + B\sin(\omega_d t)\right]$$

Osilasi dengan amplitudo yang meluruh secara eksponensial.

Critically Damped ζ = 1

$$x(t) = (A + Bt)e^{-\omega_n t}$$

Kembali ke setimbang paling cepat tanpa berosilasi.

Overdamped ζ > 1

$$x(t) = Ae^{s_1 t} + Be^{s_2 t}$$

Kembali ke setimbang secara lambat tanpa osilasi.
